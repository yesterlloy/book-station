# BookStation - 小说阅读网站

一个高性能、全自动的现代化小说阅读网站，支持高并发访问，集成自动爬虫，简洁无广告的阅读体验。

## ✨ 特性

### 🚀 高性能架构
- **异步爬虫**：Python + Aiohttp，支持并发抓取，自动去重
- **高并发 API**：Node.js + Koa2，异步无阻塞架构
- **多级缓存**：Redis 热点数据缓存 + MongoDB 持久化存储
- **Nginx 反向代理**：静态资源缓存，负载均衡

### 📖 阅读体验
- **多主题**：日间 / 夜间 / 护眼模式
- **字体调节**：14px - 28px 无级调节
- **背景切换**：5种阅读背景色
- **行高调节**：1.4 - 2.5 倍行高
- **自动进度**：本地 + 云端双同步
- **预加载**：自动预加载下一章，无缝阅读
- **响应式**：完美适配手机 / 平板 / PC

### 🤖 自动化爬虫
- **并发抓取**：支持 10+ 并发请求
- **代理IP池**：自动切换代理，应对反爬
- **随机 UA**：100+ 浏览器 User-Agent
- **内容清洗**：自动过滤广告、垃圾文本
- **智能去重**：内容哈希去重，避免重复章节
- **增量更新**：自动检测并抓取新章节
- **断点续爬**：爬虫中断后可继续

### 👤 用户系统
- **用户注册/登录**：JWT 认证
- **书架收藏**：收藏喜欢的小说
- **阅读历史**：自动记录阅读位置
- **云端同步**：多设备同步阅读进度

## 🚀 快速开始

### 环境要求

- Node.js >= 18.x
- Python >= 3.10
- MongoDB >= 6.0
- Redis >= 7.0
- Docker & Docker Compose (推荐)

### 方式一：Docker 启动（推荐）

```bash
# 1. 克隆项目
git clone <repository-url>
cd book-station

# 2. 复制环境变量
cp .env.example .env
# 编辑 .env 文件，根据需要修改配置

# 3. 启动数据库服务
docker compose up -d mongodb redis

# 4. 启动后端服务
cd backend
npm install
npm run dev

# 5. 打开浏览器访问 http://localhost:6002
```

### 方式二：手动安装

1. **启动 MongoDB 和 Redis**

```bash
# 使用 Docker 启动
docker run -d -p 27017:27017 mongo:6
docker run -d -p 6379:6379 redis:7
```

2. **启动后端 API**

```bash
cd backend
npm install
npm run dev
```

3. **运行爬虫**

```bash
cd crawler
pip install -r requirements.txt
# 编写你的爬虫配置
# 运行爬虫...
```

4. **打开阅读器**

```bash
# 使用任意静态服务器
cd frontend/reader
python -m http.server 6002
# 访问 http://localhost:6002
```

## 📖 API 文档

### 小说相关

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/novels` | 获取小说列表 |
| GET | `/api/novels/:id` | 获取小说详情 |
| GET | `/api/novels/:id/chapters` | 获取章节列表 |
| GET | `/api/categories` | 获取分类列表 |
| GET | `/api/rank/:type` | 获取排行榜 |

### 章节相关

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/chapters/:id` | 获取章节内容 |

### 用户相关

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | `/api/auth/register` | 用户注册 | - |
| POST | `/api/auth/login` | 用户登录 | - |
| GET | `/api/auth/me` | 获取当前用户信息 | 是 |

### 书架相关

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| GET | `/api/bookshelf` | 获取书架 | 是 |
| POST | `/api/bookshelf` | 添加到书架 | 是 |
| DELETE | `/api/bookshelf/:novelId` | 移除书架 | 是 |

### 阅读历史

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| GET | `/api/history` | 获取阅读历史 | 可选 |
| POST | `/api/history/progress` | 同步阅读进度 | 可选 |

## 🎨 阅读器使用

### 打开阅读器

```
http://your-domain.com/reader/?novel=小说ID&chapter=章节ID
```

### 功能说明

1. **主题切换**：点击设置按钮，选择日间/夜间/护眼模式
2. **字体调节**：滑动滑块或点击 +/- 按钮
3. **背景颜色**：5种预设背景色
4. **自动保存**：默认开启，阅读进度自动保存
5. **目录导航**：点击目录按钮查看章节列表
6. **快捷键**：← → 翻页，空格键下一章

## 🔧 配置说明

### 后端配置 (.env)

```env
# MongoDB
MONGO_HOST=localhost
MONGO_PORT=27017
MONGO_USERNAME=admin
MONGO_PASSWORD=password
MONGO_DATABASE=bookstation

# Redis
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=redis_password

# JWT
JWT_SECRET=your-secret-key-change-in-production

# 服务端口
PORT=6001
```

### 爬虫配置

```env
# 爬虫并发数
CRAWLER_CONCURRENCY=10
# 请求延迟（毫秒）
CRAWLER_DELAY_MIN=1000
CRAWLER_DELAY_MAX=3000
# 是否启用代理
PROXY_ENABLED=false
```

## 📈 性能优化

### 缓存策略

- **章节内容**：Redis 缓存 24 小时
- **小说详情**：缓存 1 小时
- **分类/排行榜**：缓存 30 分钟
- **热点小说**：常驻内存缓存

### 数据库优化

- **索引优化**：所有查询字段建立索引
- **分页查询**：避免一次性加载大量数据
- **读写分离**：预留架构支持

### 前端优化

- **原生实现**：阅读器零依赖，快速加载
- **懒加载**：章节内容按需加载
- **预加载**：智能预加载下一章
- **本地存储**：用户设置和阅读进度本地保存

## 🛡️ 安全措施

- **JWT 认证**：无状态用户认证
- **密码加密**：bcrypt 加盐哈希
- **请求限流**：基于 Redis 的频率限制
- **数据校验**：Joi 参数校验
- **CORS 配置**：跨域访问控制
- **XSS 防护**：内容渲染自动转义

## 📊 监控与日志

- **访问日志**：所有 API 访问记录
- **错误日志**：异常堆栈追踪
- **性能监控**：响应时间统计
