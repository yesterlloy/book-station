# 前端阅读器

一个高性能、支持多主题的小说阅读器，采用原生 HTML/CSS/JS 实现，零依赖，加载速度快。

## 目录结构

```
frontend/
├── reader/                     # 阅读器页面（原生实现）
│   ├── index.html             # 阅读器入口
│   ├── css/
│   │   ├── reset.css          # 样式重置
│   │   ├── variables.css      # CSS 变量定义
│   │   ├── layout.css         # 布局样式
│   │   ├── themes.css         # 主题样式
│   │   └── components.css     # 组件样式
│   └── js/
│       ├── config.js          # 配置文件
│       ├── utils.js           # 工具函数
│       ├── api.js             # API 封装
│       └── reader.js          # 阅读器核心逻辑
└── assets/                     # 静态资源
```

## 阅读器功能特性

### 阅读体验
- ✅ **三种主题模式**：日间、夜间、护眼
- ✅ **五种背景颜色**：纯白、米黄、淡绿、羊皮纸、夜间黑
- ✅ **字体大小调节**：14px - 28px 无级调节
- ✅ **行高调节**：1.4 - 2.5 倍行高
- ✅ **平滑滚动动画**

### 阅读进度
- ✅ **自动保存进度**：本地存储 + 云端同步
- ✅ **阅读进度条**：实时显示阅读百分比
- ✅ **上次阅读位置恢复**
- ✅ **阅读历史记录**

### 导航功能
- ✅ **章节目录侧边栏**
- ✅ **上一章/下一章快速切换**
- ✅ **键盘快捷键**：←/→ 翻页，Space 下一章
- ✅ **触摸滑动翻页**：左右滑动切换章节

### 性能优化
- ✅ **预加载下一章**：阅读到 80% 时自动预加载
- ✅ **内容缓存**：API 数据缓存
- ✅ **懒加载**：按需加载章节列表
- ✅ **零依赖**：原生实现，加载速度快

### 响应式设计
- ✅ **手机端优化**：触摸友好，自适应布局
- ✅ **平板端优化**：更宽的边距，更大的字体
- ✅ **PC 端优化**：最大内容宽度限制，舒适阅读宽度

## 使用说明

### 直接打开阅读器

```bash
# 可以直接在浏览器中打开
open frontend/reader/index.html?chapter=章节ID&novel=小说ID
```

### URL 参数说明

| 参数 | 说明 | 必填 |
|------|------|------|
| chapter | 章节 ID | 否 |
| novel | 小说 ID | 否 |

如果只提供 novel ID，会自动加载第一章或上次阅读位置。

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| ← / PageUp | 上一章 |
| → / PageDown / Space | 下一章 |
| Esc | 关闭弹窗 |

### 触摸手势

| 手势 | 功能 |
|------|------|
| 左滑 | 下一章 |
| 右滑 | 上一章 |

## 主题预览

### 日间模式 (Light)
- 背景：纯白 / 米黄 / 淡绿 / 羊皮纸
- 文字：深灰黑色
- 适合：白天、光线充足环境

### 夜间模式 (Dark)
- 背景：深黑色
- 文字：浅灰色
- 适合：夜间、暗环境
- 护眼：减少蓝光，降低眼睛疲劳

### 护眼模式 (Sepia)
- 背景：暖色调羊皮纸色
- 文字：棕褐色
- 适合：长时间阅读
- 特点：色调柔和，不易疲劳

## 开发说明

### 配置修改

编辑 `reader/js/config.js` 可以调整：

```javascript
const READER_CONFIG = {
    api: {
        baseUrl: '/api',          // API 基础地址
        timeout: 10000,           // 请求超时（毫秒）
    },
    fontSize: {
        min: 14,                  // 最小字体
        max: 28,                  // 最大字体
    },
    preload: {
        threshold: 0.8,           // 阅读到 80% 时预加载
        chapters: 1,              // 预加载章节数
    },
    // ...
};
```

### 添加新主题

1. 在 `css/variables.css` 中添加主题变量
2. 在 `js/config.js` 的 themes 配置中添加新主题
3. 在 `index.html` 的主题选择器中添加按钮

### 自定义 CSS 变量

```css
:root {
    --bg-primary: #ffffff;      /* 主背景色 */
    --text-primary: #1a1a1a;    /* 主文字色 */
    --accent-color: #3b82f6;    /* 强调色 */
    --font-size-base: 18px;     /* 基础字体大小 */
    --line-height-base: 1.8;    /* 基础行高 */
    /* ... */
}
```

## 与后端集成

阅读器通过标准 REST API 与后端通信：

- `GET /api/chapters/:id` - 获取章节内容
- `GET /api/novels/:id/chapters` - 获取小说章节列表
- `POST /api/history/progress` - 同步阅读进度

API 请求格式和响应格式请参考后端 API 文档。
